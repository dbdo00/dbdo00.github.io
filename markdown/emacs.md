古人说过 

>认识就是追忆

最重要是记忆。很快忘了博客的程序怎么用，怎么添加新的文章了。现在已经很好用，随便打开一个记事本就能写、预览。顾虑也多了。 

用了几天 emacs。我从来没有用过这么好的编辑器。想说很多却发现无从下笔。

不喜欢 spacemacs 的复杂。装上两个插件就很好用。

- Cdlatex 和 auctex。[^1] vscode 不提供同样的功能，vim 的风格不同。
- 用 [emacs-bedrock](https://git.sr.ht/~ashton314/emacs-bedrock/tree/main/item/early-init.el) 作为配置的开始。
- [`reftex-mode`](https://ftp.gnu.org/pub/gnu/auctex/11.88-extra/tex-ref.pdf) 在 emacs 里导航章节 

难免沉迷缝补设置，[控制在 15 分钟就好](https://sachachua.com/blog/2024/01/choosing-what-to-hack-on/)。

为什么不用 org-mode？因为我只想写完了打印下来看，感觉用 org-mode make 的时候改错不方便。虽然 orgmode 是 emacs 的核心功能，但是用 emacs 直接写 LaTeX 也很方便。

[^1]: 我看了介绍的 [这个视频](https://youtu.be/5vYmviWKD0I) 和 [这个文章](https://blog.poi.cat/post/spacemacs-plus-org-mode-plus-latex)
