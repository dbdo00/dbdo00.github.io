i have to find that i am in the state of wanting to write or speak yet having no substential idea to express. 

last year i noticed that a warmer editor color scheme make people want to write. i used "earthsong", which is very warm. the color was like something burning or the sun. today i set up "gruvbox2" which is of longer history and more popular. 

recently i have been trying a smoother workflow for writing blog on github pages. The problem is with the editor. the method now i use is type `nvim -c 'e $blog/markdown'` in terminal and then it will open the directory of my markdown files. in neovim, i can then press `%` to create a new file. there are indeed some oddities about nvim's file manager, as a lot other parts of old advanced editors such as emacs or vim. I will talk about them and the workaround.

i also wrote a vimscript to let nvim  add, commit and push on saving a markdown file in my blog folder.
