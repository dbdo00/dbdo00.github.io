tab

vimwiki takes over tab in insert mode. This is clear because you use tab to jump to the next table entry/link.

However, you cannot use tab to trigger snippets defined by, say, Ultisnippet.

Work around :

https://github.com/vimwiki/vimwiki/issues/357#issuecomment-739437349 


let g:vimwiki_key_mappings = { 'table_mappings': 0, }

or in lua 


vim.g.vimwiki_key_mappings = { table_mappings = 0 }
