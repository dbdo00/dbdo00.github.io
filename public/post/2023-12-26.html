我想总结一下我的 2023 年。

从心理上讲，我更成熟了，事情很少超出预期。

好了，先休息一下，过会儿再来……

忏悔……这是我想要的。

我想我需要减少我的精神损失。我本可以少操点心，多做点事，但我常常因为操心而耽误了实际工作。

大多数事情都不应该成为痛苦。 这听起来很可笑；恐惧占据了大部分时间。

春天

春天不冷。 只是有一天风很大。那天以后就失去了大部分寒意。

下了一场雨（冷）。4月底很热。

夏天

夏天过得很懒。以后可能再也没机会那么懒了。舍不得睡觉，也不愿意醒来。

推特被封号了。当尝试恢复我的账号时，我感觉到自己似乎陷入了强迫症。我不断地想让客服处理我的问题，并且想要跟他们争辩和交涉。为了克服这种冲动，我试图努力集中注意力，但是却感到困惑和不确定，好像不清楚自己应该做什么。所以我每天都晚上不睡觉。推特的回应是电子邮件自动回复。

报复性地在各种 fediverse 站点注册帐号。不过不适应自说自话了，发现自己很寂寞。有一天遭到了 profile 包含民主党的一个陌生人骚扰。心理防线崩塌了。

现在想想觉得很不能理解。但是当时大概陷进去了。

甚至没有一天是心安理得的废人生活。

向网友发电子邮件（因为其他社交媒体帐号都失去了）。激动，但也迷茫，不知道自己在写什么。只是觉得对方很吸引人，抓住了一根稻草。孤独，也觉得一切都是孤独的问题，一切问题都是属于孤独。几年前的一天，断定自己得了绝症。心里很悲伤：没有人会在意。那一年我用痛苦得不行就自杀来支撑自己。

夏天开始，生病了大概半年。和用眼，作息大概都有关系。

夏天以后

意识到生活方式有问题了，我尝试改变。有句话大概是，孤独不是因为没有人在身边，而是不能充分表达自己。这句话让我有些解放。发现自己没有表达的空间，也发现公开和私人的表达不同，但是都服务于同一个目的。重新找空间写下自言自语，没有这样写过这么多。主要是写在电脑上，因为还是觉得纸上不安全。。 写过两次在纸上，都没有觉得不好，过后却没有心情再这么做。偶然有人告诉我 openai whisper 以后也经常用 whisper。

在 fediverse 上遇到一个经常转发心理学文章的人。一度对这种文章不屑一顾，当时却觉得很有帮助。我觉得不管你采取什么样的态度，分清楚不同的东西，能获得更好的心情。没有形而上学的承诺，只有应对机制。精神止痛，在当前的时间里过得更舒服一些。不过这种文章看的太多也觉得没意思和迷惑，找到点击的地方还是很好的。

吃了（止神经痛）药。结果一天到晚就想睡觉，至少睡15个小时。精力不足以后每天都过着很简单的生活。开学以后有个教授直接地对待我，有点接受不了。但是我因为不会写题，只能不停地找他。后来他似乎是尽量地对我好一点。我也很感谢他能选择这么做。

认识新朋友。

今年，我认识了一些新朋友。他们与我在上一年里交往的大多数人都不一样。在互联网上，原因大概是无心的选择和机缘，而且我花更多时间的平台也不一样了。

不仅是埃隆马斯克给 𝕏 （推特）带来变化所致。学校里，我也遇到了不同的同学和老师。

在网上和人自由交流，虽然我不能做好。我感动了。这是愉快的事。

电脑

上半年，我让 chatgpt 写了很多代码。我对自己说，我只想让它完成工作。几个月后，意识到它是多么方便，也终于明白，让它写代码浪费了我太多时间。这是欺骗，让我觉得学到了些东西。

实际上根本没有。

我不知道该如何继续，也不知道该从哪里继续。

我做的第一件事就是尝试阅读《SICP》这样的书、

很快我就放弃了。生病自然是一个主要原因，但我不太喜欢阅读的内容。

我还看了点 css 的布局和排印学。我很喜欢这些概念，但发现它们很难完全理解。

我选修了一门关于 OOP 的课程，虽然成绩不佳，但我开始欣赏它了。

还发现 javascript 的语法非常有趣。

今年年底，CS 没有什么长进。

但不知何故，我期待着更多的奉献。

今年的最后几天

我发现了一篇关于 anki 的文章。我在三月份尝试过 anki，但印象不深。这篇文章讨论了有效制作卡片的困难，以及记忆被忽视的作用。我记忆力很不好，所以能体会到。

新习惯

今年我开始做的一件新事情是阅读包括 Rss 和新闻网站在内的新闻源。

我以前从来都不喜欢 Rss。

但查看订阅的 feeds 成了我经常做的一件事。

通过为我的 Github 页面博客创建 rss feed，我学到了一些东西。

现在，我使用 inoreader 的浏览器插件，方便地订阅网站的 feed。

很多网站都隐藏了 rss 链接，但奇怪的是，他们其实提供 rss，把它们放在 alternate html 标签中（如果有更正确的叫法，请告诉我）。

我讨厌过长的文章，一开始我用 tts 听，但最近发现 chatgpt 的摘要效果更好。

hackernews 也经常访问。我不太喜欢，有些讨论和文章很难理解，读起来也很累。但正如我所说的，它们的质量吸引了我，让我在拖延时感觉好些。

结尾

倾倒完记忆，留出的空间可能可以回想起来更多的吧。到时候可以再谈谈。

开始采取戏谑和批评的态度。世界上很多制度都有值得批评的地方。制度下面的人也是批评的对象。这样想，在对一个东西不满的时候，可以看穿它，自己便能满足。虽然有些事情实在找不到批评讽刺的地方。这么想得太频繁，万能防御。有些实在找不到，就算了。

知道了一种动物，叫兔狲。我很喜欢。
